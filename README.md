# security-skillz

This is a set of agent files (and scripts generated by them) that can be used with a range of Open Source tools 

## Gemini Config

See `~/.gemini/settings.json`

```
{
  "security": {
    "auth": {
      "selectedType": "oauth-personal"
    }
  },
    "general": {
      "previewFeatures": true
  },
    "context": {
       "fileName": ["AGENTS.md","CONTEXT.md","GEMINI.md"]
     }
}
```
